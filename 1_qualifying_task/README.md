# Prediction of User Retention

Проект по предсказанию удержания пользователей (retention) с использованием ансамбля градиентного бустинга.

## Описание

Задача бинарной классификации: предсказать, вернётся ли пользователь в приложение. Используется ансамбль из трёх моделей: CatBoost, XGBoost и LightGBM с оптимизацией гиперпараметров через Optuna.
